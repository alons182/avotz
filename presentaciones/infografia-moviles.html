Porque los website para moviles importan

También podés ver esta presentación en Slideshare, haciendo click aquí

Una pequeña infografía del porqué importa que los sitios web de Costa Rica y Guanacaste sean adaptables para dispositivos móviles. El diseño web en Guanacaste y Costa Rica no ha llegado a niveles óptimos para el mejoramiento de las ventas por medio de sitios web; hay que empezar por transformar todos las páginas web para que sean responsive o multiplataforma.

Varias datos confirman que muchos usuarios con comportamientos altos de consumo inician su elección de productos/servicios por un dispositivo móvil, y además con mucha frecuencia utilizan el desktop para continuar su interacción con la marca.

Además se puede denotar que los usuarios utilizan el dispositivo móvil un 82% en apps y que el 20% es usado para navegar por páginas web que pierden gran cantidad de tráfico si los sitios web no están optimizadas para dispositivos móviles.
